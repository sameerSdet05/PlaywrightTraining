# PlaywrightTraining

## Day 2 — Simple Login Test (No POM)

### Key Concepts Learned
- `test()` – Playwright test structure
- `page` – Browser tab for automation
- `locator()` – Primary element selector
- Basic actions: click, fill, goto
- Basic assertions with `expect()`

### Commands

Run all tests:
```bash
npm test
```

Run with visible browser:
```bash
npm run test:headed
```

Debug mode:
```bash
npm run debug
```

Recorder UI:
```bash
npm run codegen
```

Open HTML report:
```bash
npm run show-report
```
